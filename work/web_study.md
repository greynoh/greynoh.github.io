## 프로그램설치: 
  sublimetext
   (반드시 컨트롤팩키지 
   <https://packagecontrol.io/installation> 를 설치한 후 사용)
    한글문제해결 - IME support
    단축키 - emmet
    markdown파일 - markdown preview/ edit
    폴더/파일관리 - sidebar 
    브라우저확인 - open in browser
    가이드라인 - indent guide
    자동경로찾기 - autofileName
    색상찾기 - color picker
    주석 - comment
    
  sublimtext단축키: 
    커멘드팔레트: 컨+시+p
    팩키지설치시: 커멘드팔레트 -> ip(install package) 
    팩키지삭제시: 커멘드팔레트 -> rep(remove package) 
    팩키지정지: 커멘드팔레트 -> dsp(disable package) 
    팩키지작동: 커멘드팔레트 -> enp(enable package) 
    sidebar 보기: 컨트롤 + k + b  
    
    emmet단축키
      텍스트감싸고 코드입력: (윈)컨트롤+ 알트 + 엔터
      세로줄 멀티 선택: (윈)컨트롤+ 알트+ 방향, (맥)쉬프트+ 컨트롤+ 방향
      라인복사: 쉬프트 + 컨트롤 + D
      기타: emmet.io 참조
